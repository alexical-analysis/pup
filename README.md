# Pup

Pup is a minimal reimplementation of the Manta programming language. It servers two primary purposes:
1) To allow developers to easily experiment with new compiler structres and features.
2) To allow develoeprs to easily understand how the Manta compiler works

It is not as fully featured as Manta but the goal is to create a minimal version useful for pototyping
and learning on a smaller codebase.

## Asside

Currently Manta needs to undergo some larger refactors so this codebase is the desired target for the
Manta compiler rather than accurately reflecting how it is currently structured.
